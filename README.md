☕ Gudy Coffee - Cardápio Digital
Um projeto prático para a digitalização do cardápio da Gudy Coffee. Este repositório contém a estrutura frontend do menu online, focado em oferecer uma visualização limpa, direta e organizada dos produtos para os clientes.

🎯 Objetivo do Projeto
Criar uma interface funcional que reflita a identidade da cafeteria, facilitando o acesso às diversas categorias do cardápio, desde os cafés especiais e chás até os croissants e toasts tradicionais. A arquitetura do código foi planejada aliando as necessidades operacionais reais de atualização do negócio com a aplicação prática de fundamentos de design de interface.

✨ Funcionalidades
Estrutura Categorizada: Produtos divididos logicamente por seções (Combos, Croissants, Tradicionais da Casa, Cafés Especiais, Bebidas Orientais, etc.).
Interface em Cards: Disposição em grid para facilitar a leitura, o escaneamento visual rápido e dar destaque às fotografias reais dos produtos.
Fácil Manutenção: Estrutura HTML/CSS construída de forma modular para permitir atualizações ágeis de preços, descrições e novos itens.
🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.
CSS3: Estilização, layout em Grid e design responsivo.
Git e GitHub: Controle de versão e documentação.
